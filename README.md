## 📊 HR Analytics Dashboard (Tableau)

## 📌 Project Overview
This project focuses on analyzing HR data to uncover key insights related to employee attrition, performance, and workforce trends. The dashboard is built using Tableau to provide interactive and visual insights for better decision-making.

## 🎯 Objective
- Analyze employee attrition patterns  
- Identify key factors affecting employee retention  
- Help HR team make data-driven decisions  

## 🛠 Tools Used
- Tableau  
- Excel (for dataset cleaning)  

## 📊 Key Insights
- High attrition observed in specific departments  
- Employees working overtime are more likely to leave  
- Job satisfaction plays a major role in retention  
- Younger employees show higher attrition rate
